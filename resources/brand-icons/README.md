# Brand icons (opzionale)

Copia qui SVG professionali (es. da Shutterstock, Freepik con licenza, o export IA)
con questi nomi per sostituire automaticamente le icone inline:

- `hair.svg` — profilo donna / parrucchiere
- `body.svg` — silhouette trattamenti corpo
- `spa.svg` — viso / benessere
- `nails.svg` — manicure
- `beauty.svg` — centro bellezza / logo mood
- `mirror.svg` — trucco / stile

Requisiti: viewBox quadrato, `fill="currentColor"` o monocromatico, senza testo embedded.
